The program is written in C++11, can be compiled with any standard C++11 compiler (I used g++).

Input:
Input is directly read from a file in the same folder as the compiled executalbe. The program takes input in the following format:
1. First line contains number of attributes including the target attribute.
2. Second line contains the zero based index of the target attribute.
3. Next line describes the type of the attributes (c = continuous, d = discrete).
4. The following lines contain comma separated attributes, one line for each sample. An empty line is used to terminate the input.
